# AiAgent Project

## Overview
AiAgent is a PHP-based platform designed to run AI agents for various tasks such as code validation and information retrieval.

## Purpose
The primary purpose of AiAgent is to provide a framework for executing AI-driven tasks. It aims to simplify the development process by offering tools for:

- **Code Validation**: Checking code quality.
- **Information Retrieval**: Searching online documentation.
- **Persistent Discussions**: Storing and continuing discussions.

## Features
- **AI Agent Management**: Run AI agents for specific tasks.
- **Code Quality Checks**: Automated validation of code.
- **Discussion Storage**: Persist and retrieve conversation history.

## Setup Instructions
1. Install dependencies: `composer install`
2. Set up environment variables in `.env` file

## Usage
- Start the application: `bin/console basic-chat-agent`
